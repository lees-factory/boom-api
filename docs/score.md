## 활동 기여도 기반 '컬러 랭킹' (Anti-Goinmul)

랭킹 철학: 난이도 그레이드가 아닌 '앱 내 활동 지수'로 랭킹 산정.

컬러 단수 시스템: 클라이밍 홀드 색상을 오마주한 계급 체계.

노랑단 / 초록단 / 빨강단 / 보라단 / 황금단

### 활동 점수(Score) 산정 로직

```
Score = (모임 참여 × 10)
      + (암장 참여횟수 × 5)
      + (피드 게시 × 3)        ← TODO: 피드 기능 확장 후 반영
      + (댓글/좋아요 × 1)      ← TODO: 피드 기능 확장 후 반영
```

| 항목 | 점수 | 상태 | 비고 |
|------|------|------|------|
| 모임(일정) 참여 | ×10 | 구현 예정 | CrewSchedule 참여 기능과 연동 |
| 암장 참여횟수 | ×5 | 구현 예정 | 입장(Admission) 기록 기반 |
| 피드 게시 | ×3 | TODO | 피드 기능 추후 확장 시 반영 |
| 댓글/좋아요 | ×1 | TODO | 피드 기능 추후 확장 시 반영 |

### 크루 온도

크루원들의 평균 활동 지수를 온도로 환산하여 크루 간 건강한 경쟁 유도.
